# Firefox

> Firefox is a digital technical product that provides information access, communication, and content interaction through the web.

## Productivty

> How to improve productivity when working in Firefox?
 
| **Task**                      | **Productivity Tip(s)**                                                                                          |
| ----------------------------- | ---------------------------------------------------------------------------------------------------------------- |
| **Tab Navigation**            | Enable MRU switching (`Ctrl+Tab`), jump via `Alt+1…9`, use `Ctrl+Shift+A` to search tabs, disable tab animations |
| **Search & Address Bar**      | Use `%` to search open tabs, `*` to match previously visited pages, enable URL suggestions for open tabs         |
| **Window Management**         | Use multiple windows for context separation; switch with `Alt+Tab`; assign workspaces (GNOME/KDE)                |
| **Developer Tools**           | Use split console (`Ctrl+Shift+K`), persistent logs, and DevTools tabs; pin DevTools to bottom for faster flow   |
| **Extensions**                | Add vertical tabs, containers, search/highlight tools, and Vim-motion extensions if useful                       |
| **Privacy / Focus**           | Use strict tracking protection, disable autoplay media, enable Reader Mode to reduce distractions                |
| **Performance**               | Disable animations (`about:config` → `browser.tabs.animate = false`), enable hardware acceleration               |
| **Bookmarking / Reading**     | Use keyword bookmarks, tag-based bookmarks, and Pocket integration if applicable                                 |
| **Downloads / File Handling** | Auto-open PDFs in viewer, set preferred file actions to reduce interruptions                                     |
| **Text / Form Productivity**  | Enable form autofill, clipboard manager extension, dictionary integration                                        |
